# Explicação básica do cálculo da folha

Este exemplo utiliza os valores exibidos no cálculo do técnico Marcelo Gonçalves Veríssimo, na competência 2026-05.

## Como o cálculo funciona

O sistema primeiro considera o total bruto produzido no mês, que, neste exemplo, foi de **R$ 5.682,07**.

Sobre esse valor, aplica-se o percentual do técnico, que neste caso é de **25%**. Isso gera a **base de cálculo da comissão**:

`R$ 5.682,07 x 25% = R$ 1.420,52`

Essa base de **R$ 1.420,52** não representa a comissão final. Ela serve para verificar se ainda existe valor disponível para comissão depois de considerados os custos fixos do técnico.

Neste exemplo, os valores fixos foram:

- **Salário base:** R$ 2.664,53
- **Vale-refeição (VR):** R$ 682,00
- **Vale-alimentação (VA):** R$ 249,00

Somando esses valores:

`R$ 2.664,53 + R$ 682,00 + R$ 249,00 = R$ 3.595,53`

Como **R$ 3.595,53** é maior do que a base de cálculo de **R$ 1.420,52**, não sobra valor para comissão.

Por esse motivo, a **comissão final do período fica em R$ 0,00**.

## Resultado deste exemplo

- **Comissão:** R$ 0,00
- **Valor pago em conta:** R$ 2.664,53
- **Valor pago em benefícios (VR + VA):** R$ 931,00
- **Total da folha:** R$ 3.595,53

## Resumo simples

Em resumo, o bruto do mês foi de **R$ 5.682,07**, o que gerou uma base de comissão de **R$ 1.420,52**. No entanto, como o salário base e os benefícios do técnico somaram **R$ 3.595,53**, não houve saldo disponível para comissão. Assim, neste fechamento, o técnico recebe o salário base em conta e os benefícios no cartão, sem comissão adicional no período.
